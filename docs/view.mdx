# View
`View` utilizes the `View of react-native`. It is a container used
for controlling the layout. In addition, `View` requires the use of a 
theme provider to be able to inject a theme. `View` also has `onRef`
property which can be used for referencing the instance of a component.

## Properties of View
| **Prop**      | **Type**                   | **Default** | **Required** |
|:--------------|:---------------------------|:------------|:-------------|
| **theme**     | Object                     | null        | Yes          |
| **classname** | String                     | ''          | No           | 
| **onRef**     | Function                   | noop        | No           | 
| **children**  | Node                       | null        | No           | 

Since View utilizes the `View of react-native`, it also inherits the props
of `View of react-native` (e.g. style). For the list of props of 
`View of react-native`, please see this 
[link](https://facebook.github.io/react-native/docs/view.html).

## Basic usage of View
The following example renders a `View` component inside another `View` component.
<Playground>
      <ThemeProvider>
            <View class="main_container" style={{backgroundColor: '#20B2AA'}}>
                  <Text>Text inside a View</Text>
                  <View class="inner_container" style={{backgroundColor: '#FFB6C1'}}>
                        <Text>Text inside another View.</Text>
                  </View>
            </View>
      </ThemeProvider>
</Playground>

## Accessing View component using an `onRef` property
To following example changes the `View` background color using the `onRef` property.
The `onRef` property is set to `{component => _view = component}`. The variable namely 
`_view` is then used to set the native property of the component and change the
background color.

<Playground>
  {() => {
      isColorPink = true;
      changeBackgroundColor = () => {
            if(isColorPink){
                  _view.setNativeProps({style:{backgroundColor: '#20B2AA'}});
                  isColorPink = false;
            }else {
                  _view.setNativeProps({style:{backgroundColor: '#FFB6C1'}});
                  isColorPink = true;
            }
      }
      return (           
            <ThemeProvider>
                  <View onRef={component => _view = component} style={{backgroundColor: '#FFB6C1'}}>
                        <Text>Hello World!</Text>
                        <Button 
                              type='navy' 
                              onPress={changeBackgroundColor}>Change Background Color
                        </Button>
                  </View>
            </ThemeProvider>
      );
  }}
</Playground>
